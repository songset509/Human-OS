# Onboarding System

**Route:** `/getting-started`

## Journey (8 steps)

1. Welcome → Dashboard
2. Core Assessment → Intelligence Center
3. Personality Mapping → Big Five
4. Goal Creation → Mission
5. Life Blueprint → Blueprint
6. AI Coach Introduction → Coach
7. First Insight Report → Insights
8. Knowledge Vault → Vault

## Features

- Progress bar (% completion)
- Milestone labels per step
- localStorage persistence (`humanos_onboarding_progress`)
- Linked from Growth Analytics

## Future
